# 俺たちダービー / Friend Derby

友だちを競走馬化して、競馬新聞風の出馬表・レース演出・号外結果画像を作るGitHub Pages向け静的アプリです。

## 使い方

1. このフォルダをGitHubリポジトリにアップロード
2. GitHub Pagesを有効化
3. `index.html` を開く

AI・サーバー・ビルド不要です。

## ファイル構成

- `index.html` 画面構造
- `style.css` 新聞風デザイン
- `data.js` 馬名・実況テンプレート
- `app.js` 生成ロジック・レース演出・PNG保存
