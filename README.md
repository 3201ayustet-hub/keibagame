# 俺たちダービー v6

## v6の修正内容

- レース進行を `requestAnimationFrame` 回数依存から `deltaTime` 方式へ変更
- レース有効時間を15秒に固定
- 5秒程度で1周してしまう問題を修正
- カットイン中はレース時間を停止
- ハプニングは progress を戻さず、その場停止に統一
- スキルは瞬間ワープではなく、一定時間だけ速度倍率アップ
- 3秒・6秒・9秒・12秒地点でイベント発生
  - 先頭側にハプニングを出しやすくする
  - 後方側にスキルを出しやすくする
- 「抜きつ抜かれつ」が見えるように順位変動通知を追加
- 終盤の順位収束は前進補正のみ
- progress は絶対に後退しない

GitHub Pagesでそのまま動く静的アプリです。
