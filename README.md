# 俺たちダービー v2

友だちを競走馬化して、しょうもない競馬新聞・周回レース・号外結果を作る GitHub Pages 向け静的アプリです。

## v2で精査・改善した点

- デフォルト出走者を3枠に整理
  - 村埜
  - 西田
  - 山田
- 特徴ボタンを押すだけで馬名・二つ名・本紙コメントが即再生成
- 馬名はカタカナのみ
- スマホ縦長向けに、横長テーブルではなく「スポーツ紙一面 + 馬柱カード」に変更
- レース演出を横移動から「馬が1周する周回トラック」に変更
- レース中にスキル発動・ハプニング・SSRスキルが発生
- 結果は号外新聞風に表示
- 出馬表・号外をPNG保存可能

## GitHub Pagesでの使い方

1. このフォルダの中身をGitHubリポジトリにアップロード
2. GitHub Pagesを有効化
3. `index.html` を開く

AI・サーバー・ビルドは不要です。

## ファイル構成

- `index.html` 画面構造
- `style.css` スマホ縦長新聞デザイン・周回トラック
- `data.js` 馬名パーツ・特徴・スキル・コメント
- `app.js` 生成ロジック・レース演出・PNG保存
