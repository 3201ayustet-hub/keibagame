# 俺たちダービー v3

じゃんけん・あみだくじ代わりに使える、ランダム順位決定レースアプリです。

## v3の主な変更

- レース開始時に順位を完全ランダムで先に決定
- 能力値や特徴は勝敗に影響しない
- レース中に一時停止してスキル・ハプニングのカットイン表示
- ゴール順が分かるように、ゴールボードと馬横ラベルを追加
- 結果発表を添付画像風の「確定ボード」レイアウトへ変更
  - 馬番
  - 馬名
  - 騎手名
  - 1着/2着/3着
  - 払戻金風のネタ表
- 新聞・号外の文字サイズを全体的に抑え、スマホ1画面で見やすく調整
- デフォルト出走者
  - 村埜
  - 西田
  - 山田
- 馬名はカタカナのみ
- AI・サーバー・ビルド不要

## GitHub Pagesでの使い方

1. このフォルダの中身をGitHubリポジトリへアップロード
2. GitHub Pagesを有効化
3. `index.html` を開く
