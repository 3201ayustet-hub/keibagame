# 俺たちダービー v4

じゃんけん・あみだくじ代わりに使える、ランダム順位決定レースアプリです。

## v4の主な変更

- 払戻金要素を削除
- レース序盤〜中盤は順位が入れ替わる演出に変更
- スキル発動で対象馬が加速
- ハプニング発生で対象馬が減速
- 現在順位ボードを追加
- ゴール時に「1着！」「2着！」のオーバーレイ表示
- 表彰台を追加
- 勝利インタビューを追加
- 称号システムを追加
- 最終順位はこれまで通り、レース開始時に完全ランダムで決定
- 終盤だけ決定済み順位へ自然に収束

## GitHub Pagesでの使い方

1. このフォルダの中身をGitHubリポジトリへアップロード
2. GitHub Pagesを有効化
3. `index.html` を開く

AI・サーバー・ビルド不要です。
